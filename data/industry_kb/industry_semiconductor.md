# 行业百科：半导体（Semiconductor）

> 用途：为 FinSight 投研助理提供半导体产业链的客观知识底稿，覆盖产业链结构、关键环节、主要技术节点、估值锚定与国产替代逻辑。本文为产业普及性内容，不针对具体公司给出投资意见。

---

## 一、行业定义与范围

### 1.1 半导体的层级
半导体广义包含：
- **集成电路（IC）**：处理器（CPU/GPU/MCU）、存储器（DRAM/NAND）、模拟芯片（电源/信号链）、ASIC/FPGA、SoC；
- **分立器件**：二极管、三极管、IGBT、MOSFET、SiC/GaN 功率器件；
- **光电器件**：LED、CIS、光通信芯片、激光雷达芯片；
- **传感器**：MEMS、磁传感、压力传感、超声等。

### 1.2 全球与中国体量
- 全球半导体产业按销售额约 6000 亿美元规模（年度数字波动较大，受周期影响显著）；
- 美国（设计、设备、材料）、台湾（晶圆代工、封测）、韩国（存储）、日本（材料、设备、封装基板）、欧洲（特定领域）形成全球分工；
- 中国大陆是最大终端市场，但**自给率较低**，国产替代是核心主线。

---

## 二、产业链结构

半导体产业链高度专业化，主流分工模式有两类：
- **IDM（Integrated Device Manufacturer）**：从设计、制造到封测一体化（典型如英特尔、三星、德州仪器、意法、英飞凌）；
- **Fabless + Foundry + OSAT 分工**：设计 → 代工 → 封测三段分离，是主流模式（典型如英伟达 + 台积电 + 日月光）。

### 2.1 上游：设备、材料与 EDA / IP

#### 2.1.1 半导体设备
按生产环节分：
- **前道（晶圆制造）**：
  - **光刻机**：DUV（KrF/ArFi）、EUV，决定先进制程能力；
  - **刻蚀机**：等离子体刻蚀；
  - **薄膜沉积**：PVD（物理气相沉积）、CVD（化学气相沉积）、ALD（原子层沉积）；
  - **离子注入机**；
  - **CMP（化学机械抛光）**；
  - **湿法清洗**；
  - **量测/检测**：缺陷检测、关键尺寸量测、套刻量测。
- **后道（封测）**：划片机、键合机、塑封机、测试机、分选机、探针台。

光刻机是产业链皇冠：DUV 主要供应商为 ASML、尼康、佳能；EUV 全球唯一供应商为 ASML。

#### 2.1.2 半导体材料
- **硅片（Wafer）**：6/8/12 英寸，纯度极高的单晶硅；
- **光刻胶**：KrF、ArF、ArFi、EUV 光刻胶等；
- **湿电子化学品**：超净高纯试剂；
- **电子特气**：光刻、刻蚀、沉积、掺杂用气体；
- **CMP 抛光垫与抛光液**；
- **靶材**：用于 PVD；
- **掩模版（Photomask）**；
- **封装基板（IC Substrate）**、**键合丝、塑封料**。

#### 2.1.3 EDA（电子设计自动化）软件
全球三巨头：Synopsys、Cadence、Siemens EDA。是芯片设计的"工业母机软件"，国产化是关键短板。

#### 2.1.4 IP 核
ARM、SiFive（RISC-V）、Imagination、CEVA 等。架构授权（CPU/GPU/NPU）+ 各类标准接口 IP（DDR、PCIe、USB、HBM、SerDes 等）。

### 2.2 中游：设计、制造、封测

#### 2.2.1 设计（Fabless）
按下游应用分：
- **数字芯片**：CPU、GPU、NPU、AP（应用处理器）、Modem、ASIC、FPGA；
- **存储芯片**：DRAM（内存）、NAND（闪存）；
- **模拟芯片**：电源管理（PMIC）、信号链（ADC/DAC、放大器）、射频前端（RFFE）；
- **专用芯片**：CIS（图像传感）、Wi-Fi/蓝牙、SoC、车规 MCU；
- **功率半导体**：IGBT、MOSFET、SiC、GaN。

#### 2.2.2 制造（Foundry / IDM Fab）
晶圆代工厂按"先进制程"与"成熟制程"分类：
- **先进制程**：≤7nm，目前能量产 5nm 与更先进制程的厂商极少（台积电、三星，英特尔追赶）；
- **成熟制程**：14nm 及以上，是模拟、电源、CIS、MCU、车规 IC 的主战场。

工艺节点的代际差异主要体现在性能、功耗、晶体管密度。从 28nm 起被认为进入"先进逻辑工艺"。

#### 2.2.3 封测（OSAT）
- **传统封装**：DIP、SOP、QFP、BGA；
- **先进封装**：FCBGA、FOWLP/FOPLP（扇出型）、SiP（系统级封装）、CoWoS（台积电先进封装平台）、HBM 集成、3D 堆叠、Chiplet；
- 先进封装是当下产业链最热的细分之一，AI 算力芯片对 CoWoS 与 HBM 的依赖度急剧提升。

### 2.3 下游应用
- **消费电子**：手机、PC、平板、可穿戴；
- **汽车电子**：新能源车的电控、智驾域控、座舱、车规 MCU、SiC；
- **数据中心 / 服务器**：CPU、GPU、HBM、网络芯片、电源；
- **工业控制 / 物联网**；
- **通信基础设施**：基站、光通信；
- **国防与航天**。

## 三、关键技术与趋势

### 3.1 摩尔定律的演进
- 经典摩尔：每 18~24 个月晶体管密度翻倍；
- 当前已接近物理极限，依赖**EUV、High-NA EUV、GAA（环绕栅极）、3D 堆叠、Chiplet** 等延续；
- 2025 年前后量产节点：3nm（已量产）、2nm（GAA 架构，2025~2026）。

### 3.2 先进封装与 Chiplet
- **CoWoS**：台积电主导的 2.5D 封装平台，AI GPU 与 HBM 集成的核心载体；
- **Chiplet**：将单一大芯片拆为多个小芯粒，降低良率成本、提高灵活性；
- **HBM（高带宽内存）**：GPU 旁紧贴堆叠的 DRAM，是 AI 算力关键瓶颈，主要供应商为 SK 海力士、三星、美光。

### 3.3 第三代半导体
- **SiC（碳化硅）**：高压、高温、高频，新能源车 800V 平台、光伏逆变器、储能；
- **GaN（氮化镓）**：高频、快充电源、毫米波通信；
- **GaO / 金刚石**：早期研究阶段。

### 3.4 国产替代的核心点
- 设备：光刻、刻蚀、薄膜、量测；
- 材料：光刻胶、电子特气、靶材、抛光液；
- EDA：全流程 EDA 工具的国产对标；
- 制造：先进制程的代工能力；
- 设计：高端 GPU、高端 CPU、高端模拟芯片；
- 存储：DRAM 与 NAND 的中国本土自给。

### 3.5 周期性
半导体是典型的周期性行业，主要驱动：
- **需求周期**：手机 / PC / 服务器 / 汽车的下游景气；
- **库存周期**：约 12~18 个月；
- **资本开支周期**：3~5 年；
- **技术周期**：先进制程切换（约 2 年一代）。

## 四、产业链利润分配与估值锚定

### 4.1 不同环节的盈利特性
- **设备与材料**：高毛利但项目制周期长，订单驱动；
- **EDA / IP**：高毛利、强复利，软件商业模式；
- **晶圆代工**：重资本、高 CAPEX，规模与制程领先才有定价权；
- **设计（Fabless）**：轻资产、强研发，毛利率与产品路线高度相关；
- **存储**：强周期，价格波动巨大；
- **模拟芯片**：长尾、高粘性、产品生命周期长（10~20 年），毛利率高且稳定；
- **封测**：相对薄利，但先进封装利润率显著高于传统封装。

### 4.2 主流估值方法
- **设计公司（Fabless）**：PE / PEG / PS（亏损期）+ 产品线增长；
- **晶圆代工**：PB + 产能利用率 + 制程结构 + EV/EBITDA；
- **设备 / 材料**：PE + 在手订单 / 验证进度；
- **EDA**：PE + 收入增速 + 续费率；
- **存储周期股**：PB + 周期阶段。

### 4.3 关键观察指标
- 月度全球半导体销售额（SIA / WSTS 公布）；
- 北美半导体设备出货金额（SEMI 月报）；
- 台积电、三星、SK 海力士的财报与展望；
- DRAM 现货价、NAND 现货价；
- 各家晶圆厂稼动率（Utilization Rate）；
- 资本开支指引（CAPEX Guidance）。

## 五、风险与不确定性

### 5.1 行业层面
- **景气周期下行**：消费电子需求疲弱、库存高企；
- **地缘政治**：美国对华先进制程 / 设备 / EDA 出口管制持续升级；
- **技术路线切换**：先进封装、Chiplet、HBM 等方向变化；
- **设备国产化进度低于预期**：影响国产产能爬坡。

### 5.2 公司层面
- 客户集中度高 / 单一项目依赖；
- 高 CAPEX 折旧压力；
- 良率与制程切换风险；
- 出口管制与合规风险。

---

## 六、关键政策

### 6.1 中国
- **国家集成电路产业投资基金（"大基金"）**：一期、二期、三期已设立，定向支持设备、材料、设计、制造；
- **税收优惠**：集成电路企业减免企业所得税、研发费用加计扣除等；
- **科创板**：半导体公司是科创板核心赛道之一。

### 6.2 海外
- **美国 CHIPS and Science Act**：520 亿美元补贴本土半导体制造与研发；
- **美国对华出口管制**（BIS Entity List、先进计算与半导体管制规则）：限制 EUV、先进 DUV、AI GPU、设备与 EDA 对中国销售；
- **欧洲芯片法案**（European Chips Act）：430 亿欧元投入，目标 2030 年欧洲产能份额翻倍；
- **韩国 K-CHIPS 法案**、**日本对台积电熊本厂 / 美光投资支持**等。
