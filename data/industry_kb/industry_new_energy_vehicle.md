# 行业百科：新能源汽车（NEV）

> 用途：为 FinSight 投研助理提供新能源汽车产业链的客观知识底稿，覆盖产业链结构、关键环节、主要技术路线、估值锚定与风险点。本文为产业普及性内容，不针对具体公司给出投资意见。

---

## 一、行业定义与范围

### 1.1 新能源汽车的口径
中国官方口径下的"新能源汽车"主要包括：
- **纯电动汽车（BEV）**：仅依靠电池驱动；
- **插电式混合动力汽车（PHEV）**：可外接充电，兼具电池与发动机；
- **增程式电动汽车（EREV）**：发动机不直接驱动车轮，仅作为发电机为电池或电机供电；
- **燃料电池汽车（FCEV）**：以氢燃料电池为能源（行业初期阶段）。

### 1.2 全球与中国市场体量
- 中国是全球最大的新能源车单一市场，渗透率（新能源车销量 / 乘用车总销量）从 2020 年的约 5% 上升到 2024 年的 40%+，部分月份突破 50%；
- 全球新能源车销售大国还包括欧洲（德国、英国、法国、挪威）、美国、东南亚等；
- 出海是中国整车与产业链 2024~2025 年的核心增量主线之一。

## 二、产业链结构

新能源汽车产业链通常按"上游资源 → 中游材料零部件 → 下游整车"的链路梳理，核心三大子系统是 **三电系统（电池、电机、电控）**。

### 2.1 上游：资源与原材料
- **锂资源**：锂矿石（澳洲为主）、盐湖卤水（中国青海/西藏、智利、阿根廷）；
- **镍 / 钴 / 锰**：印尼、菲律宾（镍）、刚果（钴）；
- **石墨 / 硅基负极原料**；
- **磷酸铁原料 / 六氟磷酸锂等电解液原料**；
- **铜箔 / 铝箔基础原料**。

锂、镍、钴价格周期是新能源车产业链利润分配的核心变量。

### 2.2 中游：材料与零部件

#### 2.2.1 动力电池（电芯）
按正极材料分两条主流路线：
- **三元锂（NCM/NCA）**：能量密度高，成本高，热稳定性较差，多用于中高端长续航；
- **磷酸铁锂（LFP）**：能量密度较低但循环寿命长、成本低、安全性好，2021 年起在中国市场份额反超三元，到 2024 年中国市场动力电池中 LFP 占比超过 70%。

按封装方式分：
- **方形电芯**（中国主流）；
- **软包电芯**（部分车型 / 储能）；
- **圆柱电芯**（特斯拉 4680、储能与小动力为主）。

电池系统层级：电芯 → 模组（Module）→ 电池包（Pack）。新趋势是 **CTP（Cell-to-Pack）** 与 **CTC（Cell-to-Chassis）**——直接跳过模组、甚至取消独立 Pack，提高成组效率与体积利用率。

#### 2.2.2 电池关键材料
- **正极材料**：磷酸铁锂、三元材料、钴酸锂、锰酸锂；
- **负极材料**：人造石墨、天然石墨、硅基负极（高能量密度方向）；
- **电解液**：六氟磷酸锂为核心溶质；
- **隔膜**：湿法 / 干法基膜 + 涂覆，决定安全性与倍率性能；
- **铜箔 / 铝箔**：极薄化（4~6μm）是降本主线。

#### 2.2.3 电机与电控
- **电机**：永磁同步电机（PMSM）为主流，功率密度高、效率高；扁线电机渗透率快速提升，800V 高电压平台对应高速电机；
- **电控**：以 IGBT / SiC（碳化硅）为核心。**碳化硅**渗透是高电压平台、高效率、长续航的关键，在 800V 平台车型中价值量提升明显。

#### 2.2.4 智能化关键零部件
- **车规级芯片**：MCU、SoC、电源管理、CIS 图像传感、通信芯片；
- **激光雷达 / 毫米波雷达 / 摄像头**；
- **域控制器（智驾域 / 座舱域 / 车身域 / 动力域）**：从分布式 ECU 向中央计算演化；
- **线控底盘**：线控制动（EHB / EMB）、线控转向（SBW）；
- **HUD、AR-HUD、智能座舱大屏、车机芯片**。

#### 2.2.5 充换电基础设施
- **公共充电桩 / 私桩**：交流慢充、直流快充、超充（≥ 250kW）；
- **换电站**：以蔚来、宁德时代为代表的方案，主打 B 端运营车与高端乘用车；
- **车网互动（V2G）**：仍处早期。

### 2.3 下游：整车与服务
- **造车新势力**：理想、小鹏、蔚来、零跑等；
- **传统车企新能源转型**：比亚迪（已全面新能源化）、吉利、长城、上汽、广汽、长安、奇瑞等；
- **跨界与生态**：华为系（鸿蒙智行：问界、智界、享界、尊界）、小米（SU7 / YU7）、合资品牌的新能源转型；
- **海外**：特斯拉、大众 ID 系列、福特电马、Stellantis、丰田、现代起亚等；
- **后市场**：保险、二手车残值、维修、电池回收。

## 三、关键技术与趋势

### 3.1 电池技术演进
- **磷酸锰铁锂（LMFP）**：在 LFP 基础上提升能量密度；
- **半固态 / 固态电池**：核心方向是用固态电解质替代液态电解液，提升能量密度与安全性，量产时间表为 2025~2030 年逐步推进；
- **钠离子电池**：成本低、低温性能好、资源丰富，主要应用于储能与低端动力市场；
- **快充与倍率**：3C / 4C / 5C 倍率电池配合超充网络，10 分钟充电 400~500km。

### 3.2 高电压平台
800V 高电压平台可实现更短充电时间、更小线径、更轻整车。SiC（碳化硅）功率器件在 800V 平台上经济性显著。

### 3.3 智能驾驶
- **L2 → L2+ → L3**：高速 NOA、城市 NOA、高速 / 城市领航辅助；
- 中国市场 2024 年开始进入"城市 NOA"普及期，2025 年部分车型实现"无图智驾"；
- 计算平台主流方案：英伟达 Orin / Thor、华为 MDC、地平线 J5/J6、Mobileye 等。

### 3.4 智能座舱
- 高通 8155 / 8295 主导座舱 SoC；
- 大屏化、多屏化、AI 大模型上车（语音助手、生成式回答、个性化推荐）。

### 3.5 出海与全球化
- 整车出海：欧洲、东南亚、中东、拉美；
- 产能出海：匈牙利、墨西哥、泰国、印尼建厂；
- 关税与合规：欧盟反补贴关税、美国 IRA 法案、UNECE 法规等是出海核心变量。

## 四、产业链利润分配与估值锚定

### 4.1 利润分配的周期性
- 2020~2022：上游锂、镍、钴价格暴涨，**资源端利润最丰厚**；
- 2023~2024：锂价回落（碳酸锂价格从约 60 万元/吨 高点回落至 7~10 万元/吨），**资源端利润收缩**，中游材料与电池利润修复，下游整车价格战；
- 一般规律：上游 → 中游 → 下游 → 应用，景气会沿产业链传导，但传导速度与幅度受供需弹性影响。

### 4.2 主流估值方法
- **整车**：PS（销量×单车价值）、单车净利、PE（盈利稳定后）、销量增速；
- **电池**：PE、EV/EBITDA、产能 GWh × 单 GWh 净利、客户结构；
- **材料**：PE、PB（重资产）+ 库存周期；
- **设备 / 锂电设备**：PE + 在手订单/交付节奏；
- **充电桩 / 充电运营**：DCF + 桩点利用率。

### 4.3 关键观察指标
- 月度新能源车批发 / 零售销量与渗透率（中汽协、乘联会公布）；
- 动力电池装机量（月度，电池产业创新联盟公布）；
- 碳酸锂、氢氧化锂、六氟磷酸锂等关键原料价格；
- 头部车企的车型矩阵更新与定价策略；
- 海外市场的政策变化（关税、补贴、排放标准）。

## 五、风险与不确定性

### 5.1 行业层面
- **价格战压缩盈利**：国内整车价格战延续，单车毛利率下行压力；
- **补贴退坡 / 购置税优惠到期**：政策性需求脉冲；
- **海外贸易壁垒**：欧盟反补贴关税、美国 IRA 限制、东南亚地方化要求；
- **技术路线切换**：固态电池、钠电、800V 等切换速度直接影响现有产能价值。

### 5.2 公司层面
- 销量与库存周转：库存高企叠加价格战可能引发存货跌价；
- 应收账款与回款周期；
- 海外建厂的资本开支节奏；
- 智能驾驶研发投入与变现能力的剪刀差。

## 六、关键政策与监管

### 6.1 中国
- **新能源汽车产业发展规划（2021-2035 年）**：长期路线图；
- **双积分政策（CAFC + NEV 积分）**：约束传统车企新能源化节奏；
- **购置税优惠**：阶段性免征 / 减征延续到 2025 年底，2026 年起减半；
- **补贴政策**：2022 年底退出"国补"，2024~2025 年改以"以旧换新"等形式刺激消费；
- **充电基础设施建设规划**：分省落地。

### 6.2 海外
- **美国 IRA（通胀削减法案）**：电动车税收抵免，要求电池关键矿物与组件本地化比例；
- **欧盟新车 CO2 排放法规**与**反补贴关税**；
- **UNECE R155 / R156**：网络安全与软件升级（OTA）法规。
