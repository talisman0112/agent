# 行业百科：AI 算力产业链

> 用途：为 FinSight 投研助理提供 AI 算力（含 AI 芯片、服务器、数据中心、网络互联、电力散热）的客观知识底稿。本文为产业普及性内容，不针对具体公司给出投资意见。

---

## 一、行业定义与范围

### 1.1 AI 算力的内涵
AI 算力产业链以**大模型训练与推理需求**为核心驱动，可拆为：
- **算力芯片**：GPU、ASIC（TPU、NPU 等定制 AI 芯片）、FPGA、CPU；
- **存储与互联**：HBM、SSD、网络交换芯片、Nvlink / IB 等高速互联；
- **服务器与系统集成**：AI 服务器、高性能计算（HPC）服务器；
- **数据中心基础设施**：机柜、冷却（液冷/风冷）、UPS、配电、PDU；
- **能源与电力**：变压器、高压配电、绿电、储能；
- **应用与软件栈**：模型框架、推理引擎、分布式调度、向量数据库。

### 1.2 全球与中国市场体量
- 2023~2025 年是全球 AI 算力 CAPEX 大规模投入期，云厂商（北美 hyperscaler）、主权 AI、企业内部 AI 三类主体并行扩张；
- 北美 4 大云厂商（亚马逊、微软、谷歌、Meta）合计年度 CAPEX 数千亿美元量级；
- 中国推进国产替代算力体系，叠加"东数西算"工程，对国产 GPU / 服务器 / 网络 / 数据中心需求形成结构性拉动。

## 二、产业链结构

### 2.1 芯片层

#### 2.1.1 训练芯片（Training）
- **GPU**：英伟达 H100 / H200 / B100 / B200（Hopper/Blackwell 架构），AMD MI300X / MI325X / MI355X；
- **ASIC**：Google TPU（v4/v5p/v6 Trillium）、亚马逊 Trainium、Meta MTIA、微软 Maia；
- **国产 GPU / AI 芯片**：华为昇腾系列、寒武纪、海光、燧原、壁仞、摩尔线程、沐曦等；
- 训练芯片对单芯片算力（FP16/FP8/BF16 TFLOPS）、显存容量与带宽、片间互联（Nvlink / 国产替代）要求极高。

#### 2.1.2 推理芯片（Inference）
- 云端推理：仍以 GPU 为主，部分场景使用 ASIC；
- 边端 / 端侧推理：NPU（Apple、高通、联发科）、专用 AI 加速器；
- "推理"市场规模长期看会显著大于训练。

#### 2.1.3 关键算力指标
- **算力**：FP32 / FP16 / BF16 / FP8 / INT8 等数据精度下的 TFLOPS / TOPS；
- **显存容量与带宽**：HBM3 / HBM3E / HBM4 决定大模型可训练规模与上下文长度；
- **片间互联**：Nvlink、NVSwitch、InfiniBand、UltraEthernet、国产协议；
- **能效比（TOPS/W）**：决定单位算力的电费与冷却成本。

### 2.2 存储与互联

#### 2.2.1 HBM（高带宽内存）
- **作用**：紧贴 GPU 堆叠的 DRAM，解决 GPU "喂料"瓶颈；
- **规格演进**：HBM2 → HBM2E → HBM3 → HBM3E → HBM4；
- **主要供应商**：SK 海力士（领先）、三星、美光（追赶）；
- **集成方式**：通过 CoWoS 等先进封装与 GPU/AI 芯片集成。

#### 2.2.2 网络互联
- **InfiniBand**：英伟达 Mellanox 主导，HBM 时代 GPU 集群标配；
- **以太网（UltraEthernet）**：北美超大规模厂商联盟（UEC）推动以太网替代 IB；
- **Nvlink / NVSwitch**：英伟达 GPU 间高速直连；
- **国产替代**：华为 UB、海光"海互"等自研协议。

#### 2.2.3 存储
- **企业级 SSD**：3D NAND 大容量化（QLC、PLC），AI 训练数据集与 checkpoint；
- **DRAM**：服务器内存（DDR5、CXL 内存扩展）；
- **CXL**：连接 CPU、GPU 与外部内存池的开放标准。

### 2.3 服务器与系统层

#### 2.3.1 AI 服务器架构
- 单机 8 卡 GPU 配置（如 HGX H100 8-GPU）是当前主流；
- 模块化整机柜（NVIDIA DGX SuperPOD、GB200 NVL72 等）将多个节点整合为机柜级超节点；
- 国产 AI 服务器：华为昇腾、海光、寒武纪等的整机方案。

#### 2.3.2 服务器供应商
- 全球：戴尔、HPE、超微（Supermicro）、联想；
- 中国：浪潮、新华三、华为、中科曙光、紫光股份等；
- ODM：广达、富士康（鸿海）、纬颖、纬创等"白牌"承接超大规模厂商订单。

### 2.4 数据中心基础设施

#### 2.4.1 制冷
- **风冷**：单机柜功率密度通常 ≤ 30kW；
- **液冷**：冷板式、浸没式两类，机柜功率密度 30~100kW+；
- AI 算力数据中心几乎必须液冷化，2024~2025 年是渗透率跃升期；
- 液冷涉及 CDU（冷却液分配单元）、Manifold、快接头、冷却液、冷却塔等。

#### 2.4.2 配电与电力
- **变压器**：中低压变压器、干式 / 油浸式；
- **UPS / 电池**：保障供电连续性；
- **PDU / 母线槽**；
- **柴油发电机 / 燃机**：备用电源；
- **储能 / BESS**：参与峰谷调节、提升能效。

#### 2.4.3 IDC 与 Colo
- **超大规模数据中心（Hyperscale）**：北美云厂商自建；
- **第三方 IDC（Colocation）**：中国主流 IDC 上市公司服务于互联网与大模型公司；
- **东数西算**：中国 8 大算力枢纽节点 + 10 个集群，引导算力向西部清洁能源富集地区集中。

### 2.5 软件与工具链
- **训练框架**：PyTorch（事实标准）、TensorFlow、JAX、MindSpore、PaddlePaddle；
- **分布式**：Megatron-LM、DeepSpeed、ColossalAI、FSDP；
- **推理引擎**：vLLM、TensorRT-LLM、SGLang、Triton；
- **集群调度与编排**：Kubernetes + GPU Operator、Slurm；
- **MLOps / LLMOps**：实验跟踪、版本管理、向量数据库、Agent 框架。

### 2.6 应用层
- **基础大模型厂商**：OpenAI、Anthropic、Google DeepMind、xAI、Meta，国内的智谱、Moonshot、深度求索（DeepSeek）、阿里通义、字节、百度文心、商汤、MiniMax 等；
- **垂直应用**：编程辅助、客服、办公生产力、教育、医疗、工业、金融；
- **Agent 与工具**：基于 LLM 的多步骤工具调用，是 2024~2025 年最热的应用形态之一。

## 三、关键技术与趋势

### 3.1 算力消耗规律
- **训练**：参数规模 × 训练 token 数 × 单步浮点开销，万亿参数级模型训练所需算力（FLOPs）极高；
- **推理**：随用户规模线性增长，长文本、多模态、推理（Reasoning）模型显著提高单 query 算力消耗；
- **测试时计算（Test-time Compute）**：OpenAI o 系列、DeepSeek-R 系列等推理模型在推理阶段消耗大量算力，重塑推理芯片需求结构。

### 3.2 大模型趋势
- **MoE（Mixture of Experts）混合专家**：以稀疏激活降低训练 / 推理算力需求；
- **多模态**：图像、视频、音频、3D；
- **上下文扩展**：从 8K / 32K 到 100 万 token 量级；
- **小模型与端侧 AI**：终端设备部署 1B~10B 参数模型，强化隐私与低延迟。

### 3.3 数据中心走向
- 单座 IDC 总功率从百兆瓦级走向**吉瓦级**（GW-scale）；
- 单机柜功率从 5~10kW 走向 50~100kW+；
- 液冷渗透率快速提升；
- 选址逐步向"低电价 + 低 PUE + 临近清洁能源"迁移。

### 3.4 国产替代主线
- **国产 AI 芯片**生态：算力卡 + CUDA 替代（华为 CANN、海光 DCU、摩尔线程 MUSA 等）；
- **国产服务器整机 / 整机柜**；
- **HBM、先进封装、设备材料**逐步替代；
- **CUDA 生态依赖**仍是结构性挑战。

## 四、产业链利润分配与估值锚定

### 4.1 各环节的"含金量"
- **GPU / AI 芯片**：英伟达占据绝对话语权，毛利率显著高于产业链其他环节；
- **HBM**：高度供不应求，三大存储厂利润核心；
- **先进封装（CoWoS）**：台积电独家平台，扩产节奏决定上游兑现；
- **AI 服务器**：毛利率较低但量大，订单弹性最强；
- **液冷与配电**：增量空间显著但单一环节天花板有限；
- **IDC 运营商**：商业模式重资产，估值锚定 EV/EBITDA + 上架率。

### 4.2 主流估值方法
- **GPU / AI 芯片设计**：PE + PEG + 一致预期；
- **服务器整机**：PE + 订单可见度 + 客户结构；
- **HBM**：PB（周期）+ EV/EBITDA；
- **IDC**：EV/EBITDA + 上架率 + DCF；
- **液冷 / 电力 / 配电**：PE + 订单 + 在手项目交付；
- **AI 应用 / SaaS**：PS / DCF / ARR 增速。

### 4.3 关键观察指标
- 北美四大云厂商（AWS、Azure、Google、Meta）的季度 CAPEX 与指引；
- 英伟达、台积电、SK 海力士、AMD、Broadcom 的财报与展望；
- 中国"东数西算"重大项目与运营商招标；
- 主流大模型的发布节奏与上下文 / 多模态能力跃迁；
- 单 GPU 训练成本、单 token 推理成本走势。

## 五、风险与不确定性

### 5.1 行业层面
- **CAPEX 周期下行**：云厂商 CAPEX 增速放缓 / 挪移；
- **地缘政治与出口管制**：美国对华 GPU 出口限制升级，影响国产链与海外链不同环节；
- **技术替代**：训练范式（MoE、量化、稀疏化）改变可能降低单位训练算力需求；
- **电力瓶颈**：北美电网容量与变压器交付周期成为新瓶颈。

### 5.2 公司层面
- 客户集中度（单一云厂商订单占比过高）；
- 制程切换 / 良率不达预期；
- 国产链仍依赖海外设备与材料的不确定性。

## 六、关键政策

### 6.1 中国
- **东数西算工程**：2022 年启动，构建全国一体化算力网络；
- **新一代人工智能发展规划**；
- **国产算力相关产业补贴 / 政府投资基金**；
- **数据要素化政策**与公共数据授权运营。

### 6.2 海外
- **美国 BIS 出口管制**：限制 H100 / H200 / Blackwell 等高端 GPU 对华销售，并波及部分服务器整机；
- **CHIPS Act**：补贴本土制造与封装；
- **欧盟 AI Act / 数据法案**：对 AI 应用的合规要求。
