# 三大财务报表速查手册

> 用途：让 FinSight 助理能够回答「资产负债表里的应付账款是什么」「现金流量表的间接法怎么看」「合同负债代表什么」等三大报表项目级问题。覆盖中国会计准则（CAS）下的常见科目，与国际财务报告准则（IFRS）大体相通。

---

## 一、资产负债表（Balance Sheet）

资产负债表是某一**时点**的财务状况快照，遵循恒等式：**资产 = 负债 + 所有者权益**。

### 1.1 流动资产（Current Assets）

#### 货币资金
现金、银行存款、其他货币资金（外汇、银行汇票保证金、信用证保证金等）。
**关注**：受限货币资金（保证金、冻结款）需在附注中查看，扣除后的可用现金才能与短期借款做覆盖比较。

#### 交易性金融资产
以公允价值计量且变动计入当期损益的金融资产，主要是为短期获利持有的股票、债券、基金。**变动直接影响利润**。

#### 应收票据 / 应收账款 / 合同资产
**应收票据**：客户开具但未到期的商业承兑或银行承兑汇票。
**应收账款**：销售商品/提供服务后形成的债权。
**合同资产**：已经履约但尚未取得无条件收款权的部分（与应收账款的区别在于**还附有其他条件**）。
**关注**：账龄结构、坏账计提比例、单一大额客户集中度——长账龄占比上升通常预示回款恶化。

#### 应收款项融资
银行承兑汇票或信用证为代表、被分类为以公允价值计量且变动计入其他综合收益的应收款项。"准现金"属性较强，通常用于供应链贴现。

#### 预付款项
向供应商提前支付但尚未收到货物/服务的款项。**关注**：预付款项异常增加可能为关联方占用资金或行业景气预期变化。

#### 存货
原材料、在产品、库存商品、发出商品、低值易耗品。
**关注**：存货周转天数与存货跌价准备；高存货+下游疲软通常预示**存货跌价**计提风险。

### 1.2 非流动资产（Non-current Assets）

#### 长期股权投资
对联营、合营、子公司的长期投资。
- 子公司：合并报表层面抵消；
- 联营/合营：权益法核算，按持股比例分享利润。

#### 固定资产 / 在建工程
**固定资产**：已投入使用的房屋、机器、设备等。**在建工程**：尚未完工或未达到可使用状态的工程。
**关注**：在建工程长期挂账不转固，可能是为延后折旧而美化利润；行业产能扩张周期的集中"在建工程→固定资产"转换会带来折旧压力。

#### 使用权资产 / 租赁负债
新租赁准则（IFRS 16 / CAS 21 修订）下，所有租赁均上表，承租方确认**使用权资产**与**租赁负债**。零售、连锁、航空等行业受影响巨大。

#### 无形资产 / 商誉 / 开发支出
**无形资产**：专利、商标、土地使用权、软件等。
**商誉**：并购溢价。**每年至少做一次减值测试**。
**开发支出**：研发支出资本化部分，达到可使用状态后转入无形资产。
**关注**：商誉占净资产比例 > 30% 通常被视为减值风险较大；研发资本化比例过高会"美化"当期利润。

#### 递延所得税资产 / 负债
反映会计利润与应纳税所得额之间的暂时性差异。规模过大或集中确认/转回，会显著影响某期净利润。

### 1.3 流动负债（Current Liabilities）

#### 短期借款 / 一年内到期的非流动负债
合计反映一年内必须偿还的有息负债。**关注**：与货币资金的覆盖比。

#### 应付票据 / 应付账款
对供应商的赊购欠款。**关注**：账期是否被延长（产业链话语权增强）或缩短（链条紧张）。

#### 合同负债
新收入准则下原"预收账款"的拆分项目，代表已收钱但尚未履约的部分（典型如酒类经销商打款、SaaS 订阅预收、地产预售房款）。
**关注**：合同负债余额是收入的**领先指标**，余额持续增长通常意味着未来 1~2 季度收入加速。

#### 应付职工薪酬 / 应交税费 / 其他应付款
经营性负债，规模与业务体量相关。

### 1.4 非流动负债

#### 长期借款 / 应付债券
长期有息负债。**关注**：到期结构（一年内到期、1~3 年到期、3 年以上到期）。

#### 递延收益
未到达确认条件的政府补助等。

### 1.5 所有者权益

- **股本**：实收资本；
- **资本公积**：股票发行溢价、其他权益工具变动；
- **盈余公积**：法定 + 任意盈余公积，用于弥补亏损或转增；
- **未分配利润**：累计未分配的留存收益；
- **库存股**：公司回购但未注销/未发放的股份；
- **少数股东权益**：合并报表中不属于母公司的部分。

## 二、利润表（Income Statement）

利润表反映一段**期间内**的经营成果。

### 2.1 收入与成本

#### 营业收入
主营业务收入 + 其他业务收入。新收入准则下需在**履约时点或履约期间**按"控制权转移"确认。

#### 营业成本
与营业收入直接对应的成本（材料、人工、制造费用等）。

#### 税金及附加
消费税、城建税、教育费附加等与销售直接相关的税金。

### 2.2 期间费用

- **销售费用**：广告、促销、销售人员薪酬、运费、终端建设；
- **管理费用**：行政管理人员薪酬、办公、折旧（管理用）；
- **研发费用**：未资本化的研发支出；
- **财务费用**：利息净支出、汇兑损益、手续费。

### 2.3 其他经营性损益

- **其他收益**：政府补助等（与日常活动相关）；
- **投资收益**：联营/合营投资利润、金融资产处置；
- **公允价值变动损益**：交易性金融资产、衍生品的公允价值变动；
- **信用减值损失**：应收账款等坏账；
- **资产减值损失**：存货、固定资产、无形资产、商誉减值；
- **资产处置收益**：固定资产、无形资产处置净收益。

### 2.4 营业利润 → 净利润

营业利润 + 营业外收入 - 营业外支出 = 利润总额；
利润总额 - 所得税费用 = 净利润；
净利润 - 少数股东损益 = 归属于母公司股东的净利润（**归母净利润**）。

### 2.5 重要派生指标

- **毛利率** = （营收 - 营业成本）/ 营收；
- **三费率** = （销售费用 + 管理费用 + 财务费用）/ 营收；
- **扣非归母净利润** = 归母净利润 - 非经常性损益（政府补助、资产处置、公允价值变动等），剔除"运气性"利润后的真实经营利润。

## 三、现金流量表（Cash Flow Statement）

### 3.1 经营活动现金流（CFO）

#### 直接法主要项目
- 销售商品、提供劳务收到的现金；
- 收到的税费返还；
- 购买商品、接受劳务支付的现金；
- 支付给职工以及为职工支付的现金；
- 支付的各项税费。

**关注**：CFO 是否长期 ≥ 净利润（含金量验证）。CFO 显著小于净利润，往往说明利润中有大量未变现的应收账款、合同资产或存货堆积。

#### 净利润 → CFO 的间接法调节项
- 加：折旧摊销（非现金成本）；
- 加 / 减：资产减值损失、信用减值损失（非现金）；
- 加 / 减：投资收益（在投资活动反映）；
- 加 / 减：经营性应收（应收账款）变化；
- 加 / 减：存货变化；
- 加 / 减：经营性应付（应付账款、合同负债）变化。

### 3.2 投资活动现金流（CFI）

主要项目：
- **CAPEX**：购建固定资产、无形资产支付的现金；
- 收回投资 / 取得投资收益收到的现金；
- 投资支付的现金（金融投资 / 收购）。

**关注**：制造业的 CAPEX 周期；CAPEX 高峰意味着未来几年折旧压力。

### 3.3 筹资活动现金流（CFF）

主要项目：
- 取得借款、发行债券收到的现金；
- 偿还债务支付的现金；
- 分配股利、利润或偿付利息支付的现金；
- 吸收投资收到的现金（增发、配股、IPO）。

**关注**：CFF 长期为正而 CFO 长期为负，说明公司主要靠融资输血——典型的"伪成长"或重资产行业扩张期信号。

## 四、三表勾稽关系（财务真伪验证的核心）

1. **利润表 → 资产负债表**：净利润 → 留存收益（未分配利润）；
2. **现金流量表 → 资产负债表**：期末货币资金 - 期初货币资金 = 现金净增加额；
3. **利润表 ↔ 现金流量表**（间接法）：净利润经过非现金调整与营运资金调整 → CFO；
4. **资产 = 负债 + 所有者权益** 必须始终恒等（合并报表层面）。

异常信号示例：
- 净利润大幅增长但 CFO 持续为负，应收账款激增 → 可能为"放账款"虚增收入；
- 货币资金充裕同时持续大额融资 → 资金真实性存疑（A 股历史案例如康美药业）；
- 商誉占净资产比例高 → 后续大额减值风险。
