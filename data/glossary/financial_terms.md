# 财经术语词典：高频投研词条

> 用途：为 FinSight 投研助理提供高频财经术语的统一定义，支撑用户「什么是 PE / ROE / 自由现金流 / EPS / 股息率 / 商誉」等知识类问答的准确召回。

---

## 一、估值相关

### 市盈率（PE，Price-to-Earnings Ratio）
反映股价相对每股收益的倍数关系，公式：PE = 股价 / 每股收益（EPS） = 总市值 / 归母净利润。常见口径包括 **PE-TTM**（滚动十二个月）、**PE-LYR**（最近一个完整会计年度）、**PE-Forward**（基于一致预期未来一年）。PE 越低通常意味着估值越便宜，但在亏损或净利润为负时 PE 失效。周期股、成长股、稳定股使用 PE 时需结合景气阶段判断。

### 市净率（PB，Price-to-Book Ratio）
反映股价相对每股净资产的倍数关系，公式：PB = 股价 / 每股净资产 = 总市值 / 归属于母公司股东的净资产。PB 适合用于资产驱动型行业（银行、保险、地产、有色、钢铁），制造业重资产标的也常用 PB 做估值锚定。PB 长期低于 1 通常意味着市场对资产质量存在质疑。

### 市销率（PS，Price-to-Sales Ratio）
反映股价相对每股营业收入的倍数关系，公式：PS = 总市值 / 营业收入。PS 主要用于尚未盈利或利润波动剧烈的成长型公司（早期 SaaS、互联网平台、生物医药等），是 PE 失效场景下的常用替代估值。

### PEG（市盈率相对盈利增长率比率）
PEG = PE / 净利润增长率（百分点形式）。彼得·林奇提出，PEG = 1 通常被认为估值合理；< 1 偏低估、> 1 偏高估。适用于成长型公司的横向对比；周期股或亏损股不适用。

### EV/EBITDA（企业价值倍数）
EV = 总市值 + 总负债 - 现金及等价物；EBITDA = 息税折旧摊销前利润。EV/EBITDA 常用于跨资本结构横向对比，剔除融资差异、税率差异、折旧政策差异的影响，是并购估值与重资产行业估值的常用指标。

### 股息率（Dividend Yield）
股息率 = 每股股利 / 股价。反映纯持股每年获得的现金回报率。高股息率标的常见于公用事业、银行、能源、消费等成熟行业，是低利率环境下重要的"类债资产"配置维度。

### 自由现金流（FCF，Free Cash Flow）
公式：FCF = 经营活动现金流净额 - 资本支出（CAPEX）。代表公司在维持现有业务规模后**真正可自由支配的现金**，是 DCF 估值的核心输入，也是衡量公司"造血能力"的关键指标。FCF 长期为负的公司高度依赖外部融资。

## 二、盈利能力相关

### 每股收益（EPS，Earnings Per Share）
EPS = 归母净利润 / 期末普通股股数（或加权平均）。是股东视角最直接的盈利度量，连接利润表与股价。**摊薄 EPS** 还需考虑可转债、期权、限制性股票等潜在稀释。

### 净资产收益率（ROE，Return on Equity）
ROE = 归母净利润 / 平均归母净资产。巴菲特的核心选股指标之一。可用**杜邦分析**拆解：ROE = 销售净利率 × 资产周转率 × 权益乘数，分别对应**赚钱能力、运营效率、财务杠杆**。长期 ROE 稳定 > 15% 通常被视为优质企业。

### 总资产收益率（ROA，Return on Assets）
ROA = 归母净利润 / 平均总资产。剔除杠杆影响后的资产盈利效率。同行业 ROA 越高，资产运营越高效。

### 投入资本回报率（ROIC，Return on Invested Capital）
ROIC = 税后净营业利润（NOPAT）/ 投入资本。比 ROE 更"纯粹"，剔除非经营性收益与融资结构干扰；与加权平均资本成本（WACC）对比可判断公司是否真正创造价值（ROIC > WACC 才创造经济价值）。

### 毛利率（Gross Margin）
毛利率 = （营业收入 - 营业成本）/ 营业收入。反映产品的定价权与成本控制能力。同行业内的毛利率差距可揭示品牌溢价、规模效应或技术壁垒。

### 净利率（Net Margin）
净利率 = 归母净利润 / 营业收入。综合反映公司在剔除所有成本、费用、税金后的最终盈利转换率。

### 经营性利润率（Operating Margin）
经营性利润率 = 营业利润 / 营业收入。剔除投资收益、营业外收入等"非主营"项目，更纯粹反映主营业务盈利质量。

## 三、运营效率相关

### 资产周转率（Asset Turnover）
资产周转率 = 营业收入 / 平均总资产。反映资产创造收入的效率，是杜邦分析三大维度之一。

### 存货周转率（Inventory Turnover）
存货周转率 = 营业成本 / 平均存货。次数越多，存货管理效率越高、资金占用越少。**存货周转天数** = 365 / 存货周转率。

### 应收账款周转率（Receivables Turnover）
应收账款周转率 = 营业收入 / 平均应收账款。反映回款效率与下游话语权。**应收账款周转天数 = 365 / 应收账款周转率**，又称**收现周期**。

### 现金转换周期（CCC，Cash Conversion Cycle）
CCC = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数。负值意味着公司在产业链中拥有强势话语权（如部分零售龙头），先收钱后付款。

## 四、偿债与资本结构相关

### 资产负债率（Debt-to-Assets Ratio）
资产负债率 = 总负债 / 总资产。综合杠杆水平。不同行业差异巨大：金融、地产高，软件、消费低。

### 有息负债率
有息负债率 = 有息负债 / 总资产。剔除经营性负债（应付账款、预收款等无息项目）后更能反映"借钱"压力。

### 流动比率与速动比率
流动比率 = 流动资产 / 流动负债；速动比率 = （流动资产 - 存货）/ 流动负债。短期偿债能力指标，速动更严苛。一般认为流动比率 ≥ 2、速动比率 ≥ 1 较稳健，但行业差异较大。

### 利息保障倍数（Interest Coverage Ratio）
利息保障倍数 = EBIT / 利息费用。反映利润对利息的覆盖程度。< 1.5 通常被视为偿债压力较大。

## 五、增长与质量相关

### 同比与环比
**同比** = （本期 - 去年同期）/ 去年同期，剔除季节性，常用于跨年比较；**环比** = （本期 - 上期）/ 上期，对短期边际变化更敏感。投研中两者通常并列报告。

### 复合年增长率（CAGR）
CAGR = （末期值 / 期初值）^(1/年数) - 1。多年增速的统一衡量。

### 营收增速分解
营收增速 = 量增（销量增长）+ 价增（单价提升）。逐项判断量价驱动是判断行业景气的重要工具。

### 商誉（Goodwill）
并购溢价部分。期末进行减值测试，若被并购资产业绩不达预期则需计提**商誉减值**，直接冲减利润，是 A 股周期性"业绩雷"的重要来源。

### 资本支出（CAPEX）
购建固定资产、无形资产、其他长期资产支付的现金。CAPEX 高的行业（半导体、运营商、地产）需重点关注资本回报与折旧节奏。

### 研发投入（R&D Spending）
**研发费用率** = 研发费用 / 营业收入。科技公司核心竞争力指标。需关注**资本化与费用化**比例：资本化部分先计入无形资产再分期摊销，会"美化"当期利润。

## 六、市场与交易相关

### 市值（Market Cap）
市值 = 股价 × 总股本。**流通市值** 仅含可交易股本。市值常用于行业内规模锚定与指数权重计算。

### 自由流通股本（Free Float）
剔除大股东、董监高、长期机构持股后的可交易股本。MSCI、富时等指数使用自由流通调整后的市值。

### 换手率（Turnover Rate）
换手率 = 成交量 / 流通股本。反映股票活跃度与短期博弈热度。

### 北上资金 / 南下资金
分别指通过沪深港通**北向**进入 A 股的境外资金、**南向**进入港股的内地资金。常被视为外资观点参考。

### 融资融券余额
融资 = 看多杠杆做多；融券 = 看空借券卖出。融资余额上升通常意味市场情绪偏多。

### 限售股解禁
原始股、定增股、股权激励等在一定锁定期后解禁可流通。解禁规模与解禁日临近通常对股价产生短期压制。

## 七、宏观与行业相关

### CPI / PPI / M1 / M2
**CPI**：居民消费价格指数，反映通胀；**PPI**：工业品出厂价格指数，反映上游周期；**M1**：狭义货币（流通货币 + 单位活期），反映企业活跃度；**M2**：广义货币（M1 + 准货币），反映总流动性。M1 与 M2 剪刀差扩大常被视为流动性活化的领先信号。

### 利率走廊
中国央行通过 **MLF（中期借贷便利）**、**OMO（公开市场操作）**、**SLF（常备借贷便利）** 等工具构建利率走廊。其中 MLF 利率与 LPR（贷款市场报价利率）密切相关。

### 信用利差
信用利差 = 信用债收益率 - 同期限国债收益率。利差走阔通常反映违约预期上升或风险偏好收缩。

### PMI
采购经理指数。> 50 表示扩张，< 50 表示收缩。中国国家统计局公布**官方制造业 PMI、服务业 PMI**；财新公布**财新制造业 PMI**（更偏中小企业）。

## 八、A 股 / 港股 / 美股结构差异

### A 股交易制度要点
**T+1 交收**、**涨跌停板 ±10%**（创业板、科创板、北交所部分品种 ±20%；ST 个股 ±5%）、**1 手 = 100 股**。重要披露窗口：年报 4 月底前、半年报 8 月底前、一季报 4 月底前、三季报 10 月底前。

### 港股交易制度要点
**T+2 交收**、**无涨跌停**、**1 手股数因股票而异**（部分股票 1 手 = 1000 股或 500 股）、**做空**机制活跃。重要披露：年报、中期报告，不强制季报。

### 美股交易制度要点
**T+1 交收**（自 2024 年 5 月起从 T+2 缩短为 T+1）、**无涨跌停但有熔断**、**1 股起买**、**做空与期权**机制成熟、**业绩季 (earnings season)** 季度披露。

## 九、ESG 与新兴口径

### ESG（环境、社会、治理）
**E**（Environment）：碳排放、能耗、污染；**S**（Social）：员工、供应链、社区；**G**（Governance）：董事会、独立性、信息披露。MSCI、明晟、富时罗素等机构提供 ESG 评级。

### 碳中和与碳达峰
中国"双碳"目标：**2030 年前碳达峰、2060 年前碳中和**。涉及电力、钢铁、化工、建材等高耗能行业的能耗双控、产能置换、碳市场配额。

### CCER（中国核证自愿减排量）
基于核证减排项目（如造林、风光、甲烷利用等）发放的可交易碳信用，可在全国统一碳市场内对履约企业进行抵消，最高比例不超过 5%。
