# 估值方法手册：从相对估值到绝对估值

> 用途：为 FinSight 投研助理提供主流估值方法的统一口径，使「为什么这家公司给 X 倍 PE」「DCF 假设是什么」等问题能从语料库直接召回，而不是模型凭印象作答。

---

## 一、相对估值法（Relative Valuation）

### 1.1 市盈率法（PE 估值）
**适用场景**：盈利稳定且为正的成熟期公司，如食品饮料、家电、银行、传媒。
**核心公式**：股价 = 每股收益 × 目标 PE。
**目标 PE 来源**：① 公司自身历史中位数；② 同行业可比公司中位数；③ 行业景气阶段调整。

**常见陷阱**：
- 周期股估值时切忌用景气顶点 EPS × 高 PE（双高），否则估值严重透支；周期顶部反而应给低 PE。
- 一次性收益（资产处置、政府补助）使 EPS 失真，应使用**扣非归母净利润**口径。
- 股权激励、可转债稀释会影响摊薄 EPS，关注**全面摊薄 EPS**。

### 1.2 市净率法（PB 估值）
**适用场景**：资产驱动型行业（银行、保险、地产、有色、钢铁、航运）；周期底部、亏损或微利时 PE 失效。
**核心公式**：股价 = 每股净资产 × 目标 PB。
**关键变量**：
- 净资产质量：是否含大额商誉、递延所得税资产、应收账款长期挂账；
- 隐含 ROE：在贡献相同 ROE 的情况下，PB 与 ROE 高度正相关；
- 表外因素：金融机构需调整为"调整后净资产"口径，扣除一次性资产质量瑕疵。

### 1.3 PEG 法
**核心公式**：PEG = PE / 净利润复合增速（百分点形式）。
**适用场景**：成长性主导的中等市值公司；不适合周期股、亏损股、转型公司。
**判断口径**：PEG = 1 通常视为合理，< 1 偏低估、> 1 偏高估。
**常见陷阱**：增速选取的时间窗口（未来 1 年 / 未来 3 年 CAGR）不同，PEG 结论会大幅波动；同时市场风险偏好低时 PEG 容忍度下降。

### 1.4 PS 估值
**适用场景**：尚未盈利或利润大幅波动的公司——典型如早期 SaaS、生物医药、新经济平台。
**核心公式**：市值 = 营收 × 目标 PS。
**关键变量**：可比公司选取必须严控收入结构和毛利率档位，否则 PS 数字不可比。

### 1.5 EV/EBITDA
**核心公式**：EV / EBITDA。
**适用场景**：跨资本结构横向对比、并购估值、重资产折旧政策差异较大的行业（电信、有线、航运、电力）。
**优势**：剔除融资差异、税率差异、折旧政策差异。
**局限**：EBITDA 忽略折旧本质上忽略了资本回报要求，重资产高折旧行业不能仅看 EBITDA。

## 二、绝对估值法（Absolute Valuation）

### 2.1 现金流折现模型（DCF）
**核心思想**：公司价值 = 未来自由现金流按合适折现率折现的现值之和。
**两阶段或三阶段建模**：
1. **明确预测期**（通常 5~10 年）：逐年预测自由现金流（FCF）；
2. **永续期**：用 Gordon 增长模型计算终值（TV）：TV = FCF_(N+1) / (WACC - g)；
3. 公司股权价值 = 各期 FCF 与 TV 的折现之和 - 净有息负债 + 非经营性资产。

**核心假设变量**：
- **WACC（加权平均资本成本）**：WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-t)；
  - Re 通过 CAPM：Re = Rf + β × ERP；
  - Rd 用公司发债利率或同评级利差；
  - β 用历史回归或行业代理；
- **永续增长率 g**：通常 ≤ 长期 GDP 增速（中国 2~3%、美国 2~2.5%）；
- **再投资率与 ROIC**：内含增长 = 再投资率 × ROIC，三者一致是建模严谨性的关键。

**敏感性分析**：DCF 高度依赖 WACC 与 g，最后一定要做敏感性表（WACC 上下 1%、g 上下 0.5%）。

### 2.2 股利贴现模型（DDM）
**核心公式**：股权价值 = 未来股利按股权资本成本折现之和。
**适用**：高分红、稳定经营、低留存比例的公司，典型如银行、电信运营商、公用事业。
**变体**：① 戈登增长模型（永续等比增长）；② H-模型（双阶段增长率线性过渡）；③ 三阶段 DDM。

### 2.3 NAV / RNAV（重置净资产法）
**适用**：资产价值容易公允评估的行业——地产、煤炭、油气、金融。
**核心思路**：将公司主要资产按重置成本或市价重估，加总减去负债得到净资产价值（NAV）。地产业常用 RNAV（含未结算楼盘按合理售价折现）。

### 2.4 SOTP（分部加总估值）
**适用**：多业务、多板块综合性集团（互联网巨头、综合型央企）。
**步骤**：① 将业务按属性拆分（成熟业务 / 成长业务 / 投资资产 / 净现金）；② 各分部用最适合的估值方法分别估值；③ 加总并扣减集团折价。

## 三、行业惯用估值法对照

| 行业 | 主流估值方法 | 关键观察口径 |
|---|---|---|
| 银行 | PB + DDM | 不良率、拨备覆盖率、ROE、净息差 |
| 保险 | EV（内含价值）+ P/EV | 新业务价值（NBV）、剩余边际、投资收益率 |
| 券商 | PB | ROE、自营弹性、股市成交活跃度 |
| 地产 | NAV/RNAV、PB | 土地储备、合同负债、净负债率 |
| 白酒 / 食品饮料 | PE + DCF | 毛利率、动销率、提价能力 |
| 医药生物 | PE / PS / EV/Sales / 管线 NPV | 集采降价幅度、研发管线、出海进展 |
| 半导体设计 | PE / PS / PEG | 国产化率、营收增速、毛利结构 |
| 半导体制造（晶圆代工） | PB + EV/EBITDA + 产能利用率 | 代工节点、稼动率、CAPEX 周期 |
| 新能源车整车 | PS + 销量 × 单车利润 | 车型矩阵、毛利率、海外占比 |
| 新能源车锂电 | PE + EV/EBITDA + 产能 | 出货量、单 Wh 净利、客户结构 |
| 互联网平台 | PS / SOTP / DCF | DAU/MAU、ARPU、广告 take rate |
| 周期股（钢铁、有色、化工） | PB + 周期阶段 | 价格、库存周期、产能利用率 |
| 公用事业（电力、水务、燃气） | DCF + DDM + 股息率 | 资产基础、上网电价、补贴回收 |

## 四、估值常见错误清单

1. **可比公司选错**：把成长股与成熟股混在一起算中位数 PE，结论失真；
2. **EPS 口径不一致**：同时使用滚动 TTM 与一致预期 NTM 做比较；
3. **永续增长率超过 GDP**：DCF 出现 g > 长期 GDP 增速时，本质等同于"公司未来吃掉整个经济"，不可能；
4. **WACC 的 β 用错周期**：上市时间过短或重组后历史 β 失真，应用行业代理或调整后 β；
5. **忽视一次性损益**：扣非前后净利润差异巨大时直接用财报净利润，估值失真；
6. **忽视股权稀释**：可转债、期权、限制性股票未稀释，每股口径偏乐观；
7. **PE 与净利润增速不匹配**：高 PE 必须有高且可持续的增长支撑，否则估值难维持；
8. **周期顶部错估**：景气顶给高 PE 等于在最高点叠加最高估值，是周期股最经典陷阱。

## 五、估值的最后一公里：估值与基本面的逻辑闭环

任何估值结果都不能脱离**为什么这家公司值这个钱**的核心叙事。一个完整的估值报告至少应能回答：
1. 公司未来 3 年的**增长来源**是什么？（量、价、结构、新业务、海外）
2. 这种增长是否**可持续**？（壁垒、份额、技术节奏、竞争格局）
3. 这种增长能否**转化为现金流**？（盈利质量、回款、资本支出）
4. 在合理的 WACC 与 g 下，估值锚定在哪个水平？
5. 哪些**风险**会让估值打折？（行业、政策、公司治理、财务）

> 估值是工具，不是结论。最终落地时务必结合公司基本面与行业景气阶段做交叉验证。
