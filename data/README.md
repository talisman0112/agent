# `data/` 目录说明 · FinSight 投研助理语料库

本目录用于 FinSight 投研助理的本地 RAG 语料库。所有文件**会被** `rag/vector_store.py`（入库流水线）统一加载、结构化分块、向量化进 Chroma。

---

## 一、目录结构

```
data/
├── glossary/             # 金融与会计术语词典（已内置）
│   ├── financial_terms.md         高频财经术语 ~80 词条
│   ├── valuation_methods.md       估值方法手册
│   └── financial_statements.md    三大报表速查
│
├── industry_kb/          # 行业百科（已内置 3 个）
│   ├── industry_new_energy_vehicle.md
│   ├── industry_semiconductor.md
│   └── industry_ai_computing.md
│
├── research_reports/     # 研报（默认空目录，按需放入真实研报）
├── company_filings/      # 公司年报/季报/重大公告（默认空目录）
├── policy/               # 政策原文（默认空目录）
│
├── _demo/                # 演示用虚构样本（跑通流程后建议删除）
│   ├── sample_company_starcore_energy.md       虚构公司 999001
│   └── sample_research_report_ai_computing.md  虚构 AI 算力研报
│
└── README.md             本文件
```

## 二、文件格式与分块策略

### 2.1 推荐格式
| 优先级 | 格式 | 说明 |
|---|---|---|
| ★★★ | `.md` | 章节天然结构化（`#` `##` `###`），与 `rag/structured_chunking.py` 完美匹配 |
| ★★★ | `.txt` | 也支持中文章节标题（"第 X 章"、"一、"、"二、"等） |
| ★★ | `.pdf` | 研报 / 年报 / 公告常见格式，已支持但表格还原一般 |
| ★★ | `.docx` / `.doc` | Word 文档 |
| ★ | `.xlsx` / `.xls` | 财务数据表格，分块效果有限，建议先转成 markdown |
| ★ | `.pptx` / `.ppt` | 路演 PPT，效果有限 |

### 2.2 文件命名建议
- 公司公告：`{ticker}_{date}_{title}.pdf`，如 `300750_2024-12-30_关于回购股份方案的公告.pdf`；
- 年报：`{ticker}_{year}_annual_report.pdf`，如 `300750_2024_annual_report.pdf`；
- 研报：`{broker}_{date}_{title}.pdf`，如 `huatai_2024-08-15_宁德时代深度.pdf`；
- 政策：`{department}_{date}_{title}.pdf`，如 `csrc_2024-04-12_退市新规.pdf`。

> 文件名中**包含日期**有助于后续按时间维度筛选或回溯。

## 三、推荐数据来源（公开合规）

### 3.1 上市公司公告与年报
- **巨潮资讯网（cninfo.com.cn）**：A 股公告**官方披露平台**，免费可下载所有上市公司年报、季报、临时公告 PDF；
- **上交所信息披露 / 深交所互动易**：交易所官方披露平台；
- **港交所披露易（disclosure.hkex.com.hk）**：港股年报与公告；
- **美股 SEC EDGAR（sec.gov/edgar）**：美股 10-K、10-Q、8-K。

### 3.2 研报
- **慧博投研资讯**（部分免费）；
- **东方财富研报中心**（公开摘要 + 部分全文）；
- **券商官网投研中心**：华泰、中信、国泰君安、海通、中金等的公开研究产品；
- **机构预期**：Bloomberg / Wind / Choice 的一致预期数据（多为付费）。

### 3.3 政策
- **国务院、发改委、工信部、财政部、央行、证监会、银保监会**等**部委官网**；
- **中国政府网（gov.cn）**；
- **省级 / 市级地方政府官网**（用于地方性产业政策）。

### 3.4 行业数据
- **中汽协、乘联会**：汽车销量；
- **中国电池产业创新联盟**：动力电池装机数据；
- **SIA / SEMI / WSTS**：半导体；
- **IDC / Gartner / Canalys**：科技终端；
- **Wind / 同花顺 iFinD / 东财 Choice**（付费）。

## 四、批量下载思路

### 4.1 巨潮资讯爬虫（建议）
巨潮资讯网公告查询接口稳定且可配合规则爬取：
- 公开 HTTP 接口可按股票代码 + 公告类型批量获取公告元数据；
- 通过元数据中的 PDF 链接下载原始 PDF；
- 控制频率（建议 1~2 秒 / 请求），遵守 robots 与服务条款。

> 预计后续会在 `scripts/` 下提供 `fetch_cninfo.py` 模板（请关注 `FINSIGHT_PLAN.md` 的 P1 阶段）。

### 4.2 SEC EDGAR
- 公开提供 RSS / JSON 接口，支持按 CIK 或 ticker 查询；
- 美股注册公司全部公告 / 财报均可批量下载。

## 五、入库流程

### 5.1 增量入库（推荐）
1. 把新文件放到对应子目录；
2. 在 Streamlit 侧边栏点击「执行入库（向量化）」；
3. 已入库（MD5 已记录）的文件会自动跳过。

### 5.2 全量重建
当本地数据有结构性变化（删除、重命名、重新组织目录）时，建议全量重建：
```powershell
python scripts/rebuild_index.py
```
此脚本会：
- 清空 `db/chroma/` 与 `db/parent_store.sqlite`；
- 清空 `md5.txt`；
- 重新扫描 `data/` 全部允许后缀的文件并入库。

## 六、规模建议

| 阶段 | 建议规模 | 备注 |
|---|---|---|
| Demo 起步 | 10~30 文件，几 MB | 内置 glossary + industry_kb + _demo 已可跑通三连击 |
| 简历版 | 50~200 文件，几十 MB | 加 5~10 只标的的年报/研报、3~5 份政策 |
| 进阶版 | 500+ 文件，数百 MB | 覆盖一个完整行业的标的 + 三年财报 + 持续追加 |

> 文件量不是越多越好，**质量 > 数量**。建议优先确保入库文件**章节结构清晰、来源可追溯、有日期**。

## 七、合规与版权

- **公开披露**信息（巨潮、SEC、交易所、政府官网）可自由使用；
- **券商研报**通常带有版权与服务对象限制，**自用研究 / 学习** 一般可接受，**公开分发或商用** 须遵守研报使用条款；
- **付费数据库**（Wind / Choice / Bloomberg）数据通常**禁止外传**，请勿入库到公开仓库；
- 所有 `_demo/` 目录下的虚构数据严禁用于真实投资决策。
